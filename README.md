# tw-table
小程序表格组件，支持左右列定位，和自定义按钮

## 基于vant weapp<br/>
1.支持左右定位显示<br/>
2.支持自定义按钮操作<br/>

## 使用顺序<br/>
1.npm isntall<br/>
2.小程序编辑器 工具-> 构建npm<br/>
3.小程序编辑器 详情-> 本地设置 -> 使用npm模块（勾上）<br/><br/><br/>

注：如果执行以上操作编辑器还在报错，请关闭后重启
